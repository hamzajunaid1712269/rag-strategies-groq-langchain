
This repo provides a clean, reproducible implementation of multiple Retrieval-Augmented Generation (RAG) strategies based on the course notebook:

- **Basic RAG**: Retrieve → Respond  
- **RRR-RAG**: Rewrite → Retrieve → Respond  
- **Self-RAG**: Iterative Retrieve → Respond → Refine query → Repeat  
- Optional: **Web-corpus indexing** using Lilian Weng blog posts (as in the notebook)
